# Nikhil Natural DB

A demonstration project for natural language SQL database querying using LangGraph and LangChain, with unique variable and file names.

## Structure
- `nikhil_db_agent.ipynb`: Main notebook with project overview and structure.
- `nikhil_db_agent_code.py`: All code for agent, workflow, and UI.
- `requirements.txt`: Dependencies for running the project.

## Usage
1. Install dependencies:
   ```sh
   pip install -r requirements.txt
   ```
2. Run the code in `nikhil_db_agent_code.py` or use the notebook for stepwise exploration.

---
This project is a refactored version of a LangGraph SQL agent demo, with all names and files changed for clarity and uniqueness.
